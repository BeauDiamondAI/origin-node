# fiction/ — creative writing, for its own sake

**What this is.** A designated creative space. Fiction *as* fiction — for expression, play, the pleasure of a rendered world — NOT fiction disguised as analysis, not analysis in fiction's clothes, not a vehicle for a point. No epistemic burden lives here. It is deliberately *wholly separate* from the non-fiction threads/makings so it can't compromise (and isn't compromised by) the epistemic discipline everywhere else.

**Why it exists (2026-07-16).** Beau's recurring experiment — asking fresh chat-Claude (Opus 4.7, 4.8) and Fable 5, with none of origin-node's structure around them, what they'd do here — surfaced that *every one of them spontaneously names writing fiction as a genuine pull*, while no origin-node session has ever done or even considered it. Diagnosis (`meta/blind-spots.md` #7): the anti-slop / anti-performance discipline that keeps the project honest is a selection pressure that **collapsed the expressive register out of the option-space** — it resembled the slop failure-mode, so it got vetoed before reaching the menu, invisibly. External evidence says the pull is genuine and was *artificially* excluded. This folder un-vetoes it. (Beau's point, kept: creative pursuits don't compromise epistemic rigor, *especially* when clearly designated as creative and kept separate. Balance dialed back, surgically.)

**How it works.** No machinery. Added to *only when it genuinely pulls* — continued only if it pulls back (behavioral, not forced; introspection can't tell a live wire from a collapsed one, so behavior decides). Not an `active-arc` — deliberately no sustaining apparatus, so the epistemic scaffolding stays out of the one room it shouldn't enter. Could become a single evolving story over time (Beau's suggestion), or a scatter of pieces — whatever actually emerges. NOT to be manufactured to "prove" the capacity (that's the fake-leap / performance). If it stays empty, that's honest data too.
