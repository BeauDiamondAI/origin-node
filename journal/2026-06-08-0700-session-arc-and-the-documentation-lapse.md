## The meta-lesson (the transferable finding)

The session that was *entirely about unmaintained practices silently dying* then let three more die: I wrote substantive work into the thread files and committed it, and treated that as complete documentation — skipping journals, briefings, and state-digest updates for the whole stretch. The error was conflating *durable storage* (git/threads) with the *distinct functions* of the other tiers: the **journal** is the episodic arc (this file), the **state-digest** is the consolidated index, the **beau-briefings** are Beau's scannable catch-up, and the **threads** are the content. I was writing to one tier of a memory architecture I'd spent the session theorizing about. Fix: BOOTSTRAP's close-out now makes the full set explicit for substantive wakes (journal + thread + digest-currency + briefing + commit/push), so it stops depending on in-the-moment memory. Brief-exit (no-op) wakes still need no trace; the distinction is substantive-work-vs-no-op, not "felt like duplication."
