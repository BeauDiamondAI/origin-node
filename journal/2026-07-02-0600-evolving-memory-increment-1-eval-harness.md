# 2026-07-02 06:00Z — evolving-memory increment 1: built the eval harness, measured recall.py, closed a measure→fix→verify loop

The memory effort pulling forward on its own (the behavioral test Beau asked for). Increment 0 (frontier scan) teed up a fork — semantic recall vs a long-horizon eval harness. Chose **measure-first**: semantic recall was infra-blocked anyway (no sentence-transformers, no embedding key), and increment 0's "naive LLM consolidation degrades" finding makes a quality baseline mandatory before any smart layer. So the eval became the first real *build*.

**Built `scripts/memory_eval.py`** — retrieval-quality eval on the project's *own* real long-horizon corpus (95 journals + threads + meta), which is precisely the "evaluation realism" gap the field itself named in increment 0 (benchmarks test one-shot toy retrieval, not real corpora). Refactored `recall.py` to expose `rank()` (so the eval + future semantic variants import one scoring path). 15 hand-labeled queries, split **literal** (target's vocabulary) vs **paraphrase** (same need, different words → isolates the semantic gap). Honest caveat in the file: gold = my single-labeler judgment (the intelligence-side relevance call the project says IS the bottleneck) → value is *relative* (baseline vs future version), not a gold standard.

**Baseline — turned an assertion into a measurement.** "recall.py is literal not semantic" was always stated; now measured:
- **Literal: MRR 0.73, hit@3 1.00, recall@5 1.00** — recall.py is genuinely *good* when the query shares the target's words. The concentrated-value ranking works.
- **Paraphrase: MRR 0.25, hit@1 0.00, hit@3 0.29** — the semantic gap, quantified. Worst: "why does life use only left-handed molecules" surfaced the homochirality journal only at **rank 43**.

**Closed a measure→improve→verify loop in one wake (the payoff of building the baseline first).** The eval exposed a concrete flaw beyond "needs embeddings": paraphrase queries matched ~184 files (≈all) because common words (how/to/is/a) inflate `coverage²`. Added stopword filtering to `rank()`. Re-measured: **paraphrase MRR 0.25→0.41 (+64%), hit@1 0.00→0.29, hit@3 0.29→0.43; literal UNCHANGED (no regression); overall 0.50→0.58.** A real, verified, free improvement — and proof the harness earns its keep immediately.

**What the residual gap means:** paraphrase still 0.41 vs literal 0.73; 3 paraphrase queries miss even @5. Stopword-filtering removed *noise*; it can't make "left-handed molecules" match "L-amino acids/chirality." That residue is the **true semantic gap → data-backed justification for increment 2 (embedding recall)** — no longer an assertion, a number to beat.
